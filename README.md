# Eros

A modern static website for a fictional ad agency, built with Vite, plain HTML, CSS, and a small amount of JavaScript.

Live site: https://eros-vercel.vercel.app

## Pages

- Home
- Services
- Work
- About
- Contact
- Social Ads Playbook

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This project is ready for Vercel. After linking a Vercel account:

```bash
npx vercel --prod
```
